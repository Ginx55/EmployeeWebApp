# EmployeeWebApp

## Overview

Simple CRUD using C# ASP.NET Core Web App and SQL as database
1.	View all list of Employee
2.	View Employee
3.  Create new Employee
4.  Remove Employee
5.	Update Employee

## Prerequisites
- .NET SDK version (.NET 8.0 Long Term Support)
- System.Data.SqlClient package
